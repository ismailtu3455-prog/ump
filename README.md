# UMP - Universal Media Player

Современный универсальный медиаплеер для видео, аудио и изображений с интеграцией YouTube Music.

## Особенности

### Основные функции
- 🎬 **Воспроизведение**: Видео, аудио, изображения
- 📁 **Плейлисты**: Управление плейлистами, сохранение на диск
- 🎵 **YouTube Music**: Поиск и воспроизведение музыки из YouTube Music
- 🎧 **Режим музыки**: Упрощённый интерфейс для прослушивания музыки
- 💻 **Режим трея**: Воспроизведение в фоновом режиме с управлением из трея
- 🌙 **Темы**: Тёмная/Светлая тема
- 🌐 **Языки**: Русский/English

### Поддерживаемые форматы
- **Видео**: MP4, WebM, MKV, AVI, MOV
- **Аудио**: MP3, WAV, FLAC
- **Изображения**: JPG, PNG, WEBP

## Установка

### 1. Установка зависимостей
```bash
npm install
```

### 2. Запуск в режиме разработки
```bash
npm run electron:dev
```

### 3. Сборка приложения
```bash
npm run electron:build
```

## Использование

### YouTube Music
1. Нажмите кнопку **"Слушать онлайн"** в верхней панели
2. Введите название трека или артиста
3. Выберите трек из результатов поиска
4. Трек добавится в плейлист и начнёт воспроизводиться

### Режим музыки
1. Откройте **Настройки**
2. Включите **"Режим музыки"**
3. Наслаждайтесь упрощённым интерфейсом с обложкой альбома

### Режим трея
1. Откройте **Настройки**
2. Включите **"Слушать в трее"**
3. При закрытии приложения оно свернётся в трей
4. Управление воспроизведением доступно из меню трея

## Горячие клавиши
- `Space` - Play/Pause
- `N` - Следующий трек
- `P` - Предыдущий трек
- `F` - Полноэкранный режим
- `Ctrl+O` - Открыть файл
- `Стрелки` - Перемотка и громкость

## Технологии

- **Frontend**: React 18 + TypeScript + Redux Toolkit
- **Стили**: Tailwind CSS
- **Сборка**: Vite
- **Desktop**: Electron 28
- **YouTube Music**: ytmusic-api
