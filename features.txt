Функции Universal Media Player v2.2.0:

Воспроизведение медиа:
- Воспроизведение видео форматов: MP4, WebM, MKV, AVI, MOV, FLV, WMV, OGG
- Воспроизведение аудио форматов: MP3, WAV, FLAC, AAC, M4A
- Просмотр изображений: JPG, JPEG, PNG, WEBP, GIF, BMP
- Ассоциация файлов с системой Windows (открытие двойным кликом)
- Drag-and-drop файлов для воспроизведения
- Автоматическое определение типа медиа по расширению
- Генерация миниатюр для видео
- Отображение прогресса воспроизведения
- Поддержка Picture-in-Picture режима
- Полноэкранный режим
- Три статуса воспроизведения: играет/пауза/завершено
- Очистка последнего кадра при удалении видео

Управление воспроизведением:
- Play/Pause (Space)
- Следующий трек (N)
- Предыдущий трек (P)
- Перемотка +/- 5 секунд (стрелки влево/вправо)
- Регулировка громкости (стрелки вверх/вниз)
- Выбор скорости воспроизведения (0.5x, 0.75x, 1x, 1.25x, 1.5x, 2x)
- Зум для изображений (50% - 300%)
- Mute/Unmute
- Воспроизведение заново после завершения

Ползунок прогресса:
- Синхронизированный ползунок с линией прогресса
- Плавное обновление позиции (60 FPS)
- Перетаскивание ползунка для перемотки
- Визуальный индикатор текущей позиции
- Отображение времени при наведении

Плейлисты:
- Создание пользовательских плейлистов
- Удаление плейлистов
- Переименование плейлистов
- Системный плейлист "Недавние"
- Добавление файлов в плейлист
- Удаление файлов из плейлиста
- Перемещение файлов между плейлистами
- Избранные треки (добавление/удаление)
- Поиск по плейлисту
- Сортировка файлов в плейлисте
- Сохранение плейлистов на диск (копирование файлов)
- Загрузка сохранённых плейлистов при старте
- Открытие папки плейлистов через проводник
- Добавление папок с медиафайлами

YouTube Music (онлайн музыка):
- Поиск музыки через YouTube Music API
- Просмотр подсказок при вводе
- Воспроизведение треков онлайн
- Отображение обложек треков
- Отображение информации о треке
- Популярные поисковые запросы
- Интеграция с ytmusic-api

Скачивание видео:
- Скачивание с YouTube (через ytdl-core)
- Выбор качества и формата
- Поддержка различных контейнеров (MP4, WebM)
- Индикатор прогресса скачивания
- Автоматическое добавление в плейлист
- Поддержка других сайтов (TikTok, Vimeo, Twitter)

Интерфейс и UI:
- Кастомный заголовок окна
- Сворачиваемая боковая панель (Sidebar)
- Изменение размера боковой панели
- Адаптивная ширина панели (пропорционально окну)
- Плавные анимации открытия/закрытия панелей
- Плавные анимации модальных окон
- Уведомления (success, error, warning, info)
- Автозакрытие уведомлений через 5 секунд
- Индикатор загрузки
- Обработка ошибок воспроизведения
- Отображение текущего файла в заголовке
- Анимации переходов
- Виртуальный скроллбар

Темы и персонализация:
- Тёмная тема
- Светлая тема
- Выбор акцентного цвета (8 пресетов + кастомный)
- Регулировка прозрачности фона (25% - 100%)
- Переключение тем через настройки

Языки:
- Русский язык
- English
- Переключение языка интерфейса

Настройки окна:
- Изменение размера окна
- Режим "Поверх всех окон"
- Picture-in-Picture режим
- Настройка прозрачности
- Сохранение настроек окна
- Адаптация интерфейса при ресайзе

Горячие клавиши:
- Space — Play/Pause
- N — Следующий трек
- P — Предыдущий трек
- F — Полноэкранный режим
- Стрелка вверх — Громкость +
- Стрелка вниз — Громкость -
- Стрелка вправо — Перемотка +5с
- Стрелка влево — Перемотка -5с
- Ctrl+O — Открыть файл
- F12 — DevTools
- Ctrl+R — Перезагрузить

Интеграция с системой:
- Ассоциация файлов (MP4, WebM, MKV, AVI, MOV, MP3, WAV, FLAC, JPG, PNG, WEBP)
- Открытие файлов из проводника
- Уведомления Windows
- Автозагрузка с Windows (опционально)
- Режим трея (скрытие в трей)
- Логирование в файл

Данные и хранение:
- История просмотров (до 100 записей)
- Избранные треки
- Сохранение плейлистов на диск
- Экспорт/импорт плейлистов (JSON)
- Очистка истории
- Экспорт избранного

Производительность:
- Аппаратное ускорение (GPU)
- Vite сборка
- TypeScript
- Оптимизированный рендеринг
- Lazy loading компонентов
- Плавные анимации (60 FPS)

Технические возможности:
- Redux Toolkit для управления состоянием
- React hooks
- Context Bridge API (безопасность Electron)
- IPC коммуникация
- Копирование файлов в плейлисты
- Работа с файловой системой
- Сетевые запросы (YouTube, YouTube Music)
