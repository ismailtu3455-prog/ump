Функции Universal Media Player v4.0.0

1) Воспроизведение медиа (единый плеер)
- Единый плеер для видео, аудио и изображений.
- Поддержка локальных форматов:
  - Видео: mp4, webm, mkv, avi, mov, ogg.
  - Аудио: mp3, wav, flac, m4a.
  - Изображения: jpg, jpeg, png, webp, gif, bmp.
- Открытие файлов из:
  - системного открытия/ассоциаций,
  - диалога выбора файлов,
  - drag&drop в окно плеера,
  - drag&drop в плейлисты/списки.
- Автоопределение типа медиа по расширению.
- Загрузка и проигрывание файла с сохранением/восстановлением состояния плеера.
- Обработка ошибок воспроизведения с показом уведомления в UI.

2) Управление плеером
- Play/Pause.
- Предыдущий / Следующий трек.
- Перемотка на +/-5 секунд.
- Регулировка громкости + mute/unmute.
- Смена скорости воспроизведения: 0.5x / 0.75x / 1x / 1.25x / 1.5x / 2x.
- Полноэкранный режим.
- Picture-in-Picture (встроенный браузерный, с fallback через Electron).
- Скриншот кадра видео (PNG).
- Режимы повтора:
  - без повтора,
  - повтор плейлиста,
  - повтор одного,
  - случайный порядок.
- Воспроизведение по клику на видео.
- Для изображений: zoom 50%-300% и сброс 1:1.

3) Прогресс и навигация по медиа
- Ползунок времени (seek).
- Отображение текущего времени / длительности.
- Обновление прогресса просмотра в состоянии плейлиста.
- Безопасная обработка duration/seekable для проблемных файлов.
- Задержка перед следующим треком (в секундах) + обратный отсчет.

4) Плейлисты и файлы
- Системный плейлист "recent".
- Создание пользовательских плейлистов.
- Переименование плейлистов.
- Удаление плейлистов.
- Сворачивание/разворачивание списка плейлистов.
- Поиск по файлам активного плейлиста.
- Добавление файлов:
  - через кнопку "Добавить файлы",
  - через добавление папки,
  - через drag&drop.
- Удаление файлов из плейлиста.
- Избранное (лайк) для файла.
- При лайке файл перемещается вверх списка.
- Drag&drop файла между плейлистами.
- Drag&drop на область плейлистов (добавление в default/recent).

5) Сохранение плейлистов на диск
- Ручное сохранение плейлиста в локальную папку проекта плейлистов.
- Загрузка ранее сохраненных плейлистов при старте приложения.
- Удаление папки плейлиста из файловой системы при удалении плейлиста.
- Прогресс копирования файлов:
  - processed / total,
  - copied count,
  - возможность скрыть/показать прогресс.
- Открытие папки плейлистов из интерфейса.
- Новая функция: "Автосохранение" (переключатель в сайдбаре).
  - При включении новые локальные файлы автоматически сохраняются в выбранный плейлист.

6) Интерфейс и UX
- Кастомное окно Electron без системной шапки.
- Кастомный верхний бар с кнопками:
  - Telegram,
  - Donate,
  - GitHub,
  - Settings,
  - Minimize / Maximize / Close.
- Гибкая ширина сайдбара (resizer).
- Современный glassmorphism-стиль с анимациями.
- Tooltip по долгому наведению на иконки.
- Кастомное всплывающее меню выбора скорости (в стиле приложения).
- Экран загрузки (splash) с анимированным логотипом.
- Очередь уведомлений:
  - success/error/warning/info,
  - автозакрытие,
  - защита от дублей,
  - ограничение длины очереди.

7) Настройки приложения
- Тема: dark / light.
- Язык: RU / EN.
- Акцентный цвет.
- Прозрачность панелей в приложении.
- Режим в трее.
- Автозапуск приложения.
- Задержка между треками (секунды) + переключатель вкл/выкл.
- Горячие клавиши:
  - отдельная вкладка настройки,
  - запись новых комбинаций,
  - сброс к дефолтным.
- Автоприменение настроек во время изменения.
- Кнопка "Сохранить" для сохранения настроек на следующий запуск.

8) Горячие клавиши (по умолчанию)
- Space: Play/Pause
- P: Previous
- N: Next
- R: Toggle repeat mode
- ArrowLeft: Seek -5s
- ArrowRight: Seek +5s
- ArrowDown: Volume -
- ArrowUp: Volume +
- Ctrl+ArrowDown: Speed -
- Ctrl+ArrowUp: Speed +
- Ctrl+0: Speed 1x
- M: Mute
- F: Fullscreen
- Ctrl+O: Open files

9) Интеграция с системой (Electron)
- Single instance lock.
- Обработка открытия файлов из ОС/командной строки.
- File associations для поддерживаемых расширений.
- Работа в системном трее (show/hide).
- Управление окном через IPC.
- Context bridge preload API для безопасного доступа из renderer.
- Сборка Windows installer (NSIS).

10) Что удалено из проекта
- Полностью удалены YouTube/онлайн-функции.
- Удалены webview и legacy-режимы.
- Удалены старые/неиспользуемые слайсы и лишние зависимости.
